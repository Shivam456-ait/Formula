Class 9–12 Chemistry Important Formulas

Complete Chapter Wise Formula List

Basic Formulas

Density = Mass / Volume

Moles = Mass / Molar Mass

Number of particles = moles × 6.022 × 10²³

Percentage = (Part / Total) × 100

C₁V₁ = C₂V₂

Thermodynamics

ΔU = q + w

H = U + PV

ΔG = ΔH − TΔS

q = mcΔT

w = −PΔV

ΔS = qrev / T

Gaseous State

PV = nRT

P₁V₁ = P₂V₂

V₁/T₁ = V₂/T₂

RMS velocity = √(3RT/M)

Equilibrium

Kc = [Products] / [Reactants]

pH = −log[H⁺]

pH + pOH = 14

Electrochemistry

E°cell = E°cathode − E°anode

ΔG° = −nFE°cell

E = E° − (0.0591/n) log Q

Faraday constant = 96500 C/mol

Solutions

Molarity = Moles / Volume

Molality = Moles / kg solvent

ΔTb = Kb m

ΔTf = Kf m

π = MRT

Chemical Kinetics

Rate = k[A]^m[B]^n

t₁/₂ = 0.693/k

k = Ae^(−Ea/RT)

Solid State

Density = (Z × M) / (a³ × NA)

a = 2r (Simple Cubic)

a = 4r/√3 (BCC)

a = 2√2 r (FCC)
