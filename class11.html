Thermodynamics

ΔU = q + w

w = −PΔV

q = m × c × ΔT

H = U + PV

ΔH = ΔU + ΔnRT

G = H − TS

ΔG = ΔH − TΔS

ΔG° = −RT ln K

qv = ΔU

qp = ΔH

Equilibrium

Kc = [Products] / [Reactants]

Kp = Kc(RT)^Δn

Q = [Products] / [Reactants]

pH = −log[H⁺]

pOH = −log[OH⁻]

pH + pOH = 14

Ka = [H⁺][A⁻] / [HA]

Kb = [OH⁻][BH⁺] / [B]

Kw = [H⁺][OH⁻]

Chemical Kinetics

Rate = k[A]^m[B]^n

Rate Constant (k) = Rate / [A]^m[B]^n

t₁/₂ (First Order) = 0.693 / k

Arrhenius Equation: k = Ae^(-Ea/RT)

ln k = ln A − Ea/RT

Atomic Structure

E = −13.6 Z² / n² (eV)

c = λν

E = hν

λ = h / mv

Δx Δp ≥ h / 4π

Radius = a₀ n² / Z

States of Matter

PV = nRT

R = 0.0821 L atm mol⁻¹ K⁻¹

Density = PM / RT

rms velocity = √(3RT/M)

Graham's Law: r₁/r₂ = √(M₂/M₁)
