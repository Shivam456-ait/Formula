🔥 Thermodynamics

ΔU = q + w

w = −PΔV

q = m × c × ΔT

H = U + PV

ΔH = ΔU + ΔnRT

G = H − TS

ΔG = ΔH − TΔS

ΔG° = −RT ln K

qv = ΔU

qp = ΔH

First Law: Energy cannot be created or destroyed

⚖ Chemical Equilibrium

Kc = [Products] / [Reactants]

Kp = Kc(RT)^Δn

Reaction Quotient Q = [Products]/[Reactants]

ΔG° = −RT ln K

pH = −log[H⁺]

pOH = −log[OH⁻]

pH + pOH = 14

Kw = [H⁺][OH⁻]

Ka = [H⁺][A⁻] / [HA]

Kb = [OH⁻][BH⁺] / [B]

⚡ Ionic Equilibrium

Degree of Dissociation (α) = Dissociated / Total

Ka = Cα² / (1−α)

Kb = Cα² / (1−α)

Buffer Solution: pH = pKa + log(Salt/Acid)

Henderson Equation: pH = pKa + log([A⁻]/[HA])

⚛ Atomic Structure

E = −13.6 Z² / n² (eV)

c = λν

E = hν

λ = h / mv

Δx Δp ≥ h / 4π

Radius = a₀ n² / Z

🌡 States of Matter

PV = nRT

R = 0.0821 L atm mol⁻¹ K⁻¹

Density = PM / RT

rms velocity = √(3RT/M)

Graham's Law: r₁/r₂ = √(M₂/M₁)

Van der Waals Equation: (P + a/V²)(V − b) = RT

⚡ Chemical Kinetics (Intro)

Rate = k[A]^m[B]^n

Rate Constant k = Rate / [A]^m[B]^n

t₁/₂ (First Order) = 0.693 / k

Arrhenius Equation: k = Ae^(−Ea/RT)
